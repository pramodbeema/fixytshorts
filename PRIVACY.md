# Privacy Policy for Fix YT Shorts

**Last Updated:** November 24, 2025

## 1. Data Collection
**Fix YT Shorts** (the "Extension") does **not** collect, store, share, or transmit any personal user data.

## 2. Permissions Usage
The Extension requires specific permissions to function:

*   **`activeTab`**: Used to detect when you are on a YouTube page to enable extension features.
*   **`https://www.youtube.com/*`**: Required to modify the YouTube player interface (add buttons, overlays) and control video playback (seeking, audio track selection).

## 3. Remote Code
The Extension is fully self-contained and does not load or execute any remote code, scripts, or analytics.

For questions or support, please contact the developer via GitHub: [https://github.com/pramodbeema](https://github.com/pramodbeema)
